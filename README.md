# Scanner Management System

An internal web application for assigning, tracking and managing handheld barcode scanners used in warehouse operations.

## Project Goals

The system helps warehouse employees quickly receive an available scanner, return it after work, and allows managers to monitor all devices in real time.

## Main Features

- Employee identification using access cards
- Automatic scanner assignment
- Scanner return process
- Scanner availability tracking
- Device status management
- Assignment history
- Dashboard and reports

## Tech Stack

- Next.js
- React
- TypeScript
- Tailwind CSS
- PostgreSQL
- Prisma ORM
- React Hook Form
- Zod
- TanStack Query

## Architecture

Frontend-first development.

The backend API and database will be implemented after the UI and business workflow are completed.
